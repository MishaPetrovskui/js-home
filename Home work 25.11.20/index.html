Завдання 4 — Вкладки

HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

HTML describes the structure of a web page and originally included cues for the appearance of the document.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML.

CSS controls layout, colors, fonts, spacing and visual effects.

JavaScript

JavaScript is a programming language used to make web pages interactive.

JS allows dynamic content, animations, forms, games and more.
